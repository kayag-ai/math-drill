# Home Edu Apps

AI（Gemini）との対話を通じて構築している、家庭内学習用のミニマルなWebアプリ集です。市販のアプリにはない「今、この瞬間の子供の習熟度」に最適化した教材を、即時的にプロトタイプ化して運用しています。

## 🛠 プロジェクトのコンセプト

**Just-in-Time Learning**  
子供が「知りたい」「やりたい」と思った瞬間に、AIを利用してその場でドリルやツールを生成・提供します。

**Minimal UI**  
学習の認知的負荷を減らすため、広告や過剰な演出を排除した、本質的な機能のみのUI設計。

**Growth-Driven Update**  
子供のフィードバックや習熟度に基づき、コードの構造（難易度やルール）を動的に変更します。
